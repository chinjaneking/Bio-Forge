# Bio-Forge

AI-Powered Synthetic Biology Platform

## 部署指南

### Windows Docker 部署

1. 解压本压缩包到目标目录
2. 确保已安装 Docker Desktop
3. 在项目目录运行：
   ```bash
   docker-compose up -d
   ```
4. 访问 http://localhost:1983/docs

## 项目结构

- `app/` - 应用程序代码
- `docker-compose.yml` - Docker编排配置
- `Dockerfile` - 容器镜像构建
- `requirements.txt` - Python依赖
